# Likitha Art Website

Minimal Luxury Website for Hand-Painted Wearable Art & Fine Art.

## Tech Stack
- React (Vite)
- Framer Motion (Animations)
- React Router DOM (Navigation)
- Vanilla CSS (Styling)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Structure
- `src/components`: Reusable components (Navbar, Footer)
- `src/pages`: Page components (Home, WearableArt, CanvasArt, etc.)
- `src/data.js`: Mock data for products and artworks
- `src/index.css`: Global styles and variables

## Design System
- Fonts: Playfair Display (Serif), Inter (Sans-serif)
- Colors: Off-white (#F5F3EE), Charcoal (#333333), Gold Accents (#CBAE7E)
